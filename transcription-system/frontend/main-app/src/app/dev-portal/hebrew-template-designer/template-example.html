דוגמת תבנית Word - העתק בדיוק!

חשוב מאוד: העתק את הטקסט הבא בדיוק כמו שהוא מופיע, כולל הסוגריים המסולסלים { }

מה לכתוב בקובץ Word שלך:

כותרת עליונה: קובץ: {fileName} תאריך: {date} גוף המסמך: דוברים: {speakers} משך הקלטה: {duration} התמלול: {#blocks} {speaker}: {text} {/blocks} כותרת תחתונה: עמוד {pageNumber}

הוראות:

1. פתח Word חדש
2. הגדר את המסמך לעברית (Layout → Text Direction → Right-to-Left)
3. הוסף מספרי שורות בצד ימין אם רצוי (Layout → Line Numbers)
4. העתק את הטקסט מהמסגרת למעלה בדיוק כמו שהוא
5. שמור את הקובץ כ- .docx
6. העלה את הקובץ ב-V2 Template Test

רשימת Placeholders:

  • {fileName} - שם הקובץ
  • {date} - תאריך
  • {speakers} - רשימת דוברים
  • {duration} - משך הקלטה
  • {#blocks}...{/blocks} - לולאה עבור בלוקי תמלול
  • {speaker} - שם הדובר (בתוך הלולאה)
  • {text} - טקסט התמלול (בתוך הלולאה)
  • {pageNumber} - מספר עמוד
טיפ: אם אתה מקבל שגיאות, ודא ש:
  • כל placeholder מתחיל ב- { ונגמר ב- } (סוגריים בודדים, לא כפולים!)
  • אין רווחים מיותרים בתוך הסוגריים
  • הלולאה blocks מתחילה ב- {#blocks} ונגמרת ב- {/blocks}
