הטקס אדיטור הוא השחקן הראשי בממשק. 
 יש קודם כל לעצב אותו כמו שצריך 
 שיתאים לצבעי העמוד, אבל שייראה גם קצת כמו מסמך וורד. 
 מעין הכלאה של השניים.
